# Projeto-Pessoal-HospitalShield-Firewall-IDS-IPS-e-SIEM-para-Ambiente-Hospitalar
Recentemente desenvolvi um laboratório de segurança inspirado em ambientes hospitalares com o objetivo de simular a proteção de uma infraestrutura crítica, aplicando conceitos de segmentação de rede, monitoramento contínuo e detecção de ameaças.
# HospitalShield Network Security Lab

Projeto fictício desenvolvido para demonstrar a implementação de uma arquitetura de segurança para ambiente hospitalar.

## Tecnologias

- pfSense
- Wazuh
- Suricata
- Windows Server
- Active Directory
- VLANs
- Firewall

## Objetivos

- Segmentação de rede
- Monitoramento de eventos
- Detecção de ameaças
- Controle de acesso
- Resposta a incidentes
